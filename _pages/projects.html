Mini-CSRanking: Mini-combination of CSRankings, GoogleScholar and DBLP
Mid-term Project for Intoduction to Database 2019 Spring @ Fudan University, by Xinyi Zhou and Zuobai Zhang
A DB-based website named Mini-CSRanking for ranking computer science departments, which is inspired by the open-source website CSRankings: Computer Science Rankings. It also supports some handy functions of GoogleScholar and DBLP, such as search for papers/scholars by keywords and scan papers on specific conferences or in certain areas.

MIPS-Microprocessors: A distinctive MIPS CPU
CPU Design Project for Introduction to Computer System II 2019 Spring@ Fudan University, by Zuobai Zhang
In this project, we have utilized Verilog language to design our own MIPS-mircoprocessors, which consist of four parts: SingleCycle Processor, MultiCycle Processor, Pipeline Processor, Pipeline Processor with cache. Additionally, we have also implemented a MIPS assembler to translate MIPS assembly into machine codes.
Geo-fencing: An application of spatial indexing data structure
Final Project for Data Structure 2018 Fall @ Fudan University, by Zuobai Zhang
In this project, we are required to apply data structures to a practical scenario. Specifically, we are supposed to employ a spatial indexing data structure to solve the geo-fencing task, in which there are a great amount of insertion, deletion and query operations about the spatial relation between points and polygons.
We discuss three spatial indexing data structures and three 'point in polygon' strategies thoroughly. Then, we propose four algorithms combining these methods and compare their performances on test dataset. Results show that Grid Method outperforms other methods on the test dataset.
